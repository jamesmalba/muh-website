Deadlock Stats API

Spring Boot analytics service for Valve's Deadlock

REST service that aggregates match and hero analytics for Deadlock, Valve's upcoming MOBA. Six endpoints over a PostgreSQL backend expose player profiles, paginated match histories, hero meta stats, and a custom Win Probability Added (WPA) metric that tracks how individual objective destructions shift match outcomes.

Built with Spring Boot on Java 21 using raw JdbcClient for SQL transparency (explicit bitmask decoding, array unnesting, DISTINCT ON) rather than Hibernate. Flyway migrations, Jakarta Bean Validation, OpenAPI 3 docs, RFC 7807 error responses, structured JSON logs, and Testcontainers-backed PostgreSQL integration tests. Containerized with Docker Compose.

Java 21 • Spring Boot • PostgreSQL • Flyway • Docker • Testcontainers

Time Traveler Debugger

Frame-rewind debugger plugin for Godot

Editor plugin that snapshots every frame of game state into a circular buffer and lets you scrub backward, inspect any past frame, toggle previously hidden debug draws, and resume gameplay from any chosen moment. Inspired by Kintsugiyama's Alta Debugger for Legend of California.

Pure GDScript, ~2,400 lines across 22 files. Runtime recorder and floating editor window communicate over Godot's EngineDebugger IPC with a two-tier protocol (lightweight summaries every frame, heavy detail on demand). Captures state via reflection over PROPERTY_USAGE_STORAGE for 1:1 save-game fidelity, respawns freed nodes from their scene_file_path, and includes an adaptive heat-map timeline with event markers and column aggregation.

Godot 4.6 • GDScript • EditorPlugin • EngineDebugger IPC • Reflection

Daily Cauldron

Full-stack recipe platform

Recipe discovery and submission platform with a wizarding-newspaper aesthetic. Browse by difficulty, summon random recipes, submit multi-section forms with dynamic ingredient/step rows, and leave 5-star ratings + written reviews.
